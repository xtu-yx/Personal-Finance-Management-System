# 前端数据校验规则文档

> 制定时间：2026-05-04

| 字段名 | 所属模块 | 校验规则 |
|--------|----------|----------|
| username | 用户注册/登录 | 非空，长度 3-50 个字符 |
| password | 用户注册/登录 | 非空，长度 6-20 个字符 |
| email | 用户注册 | 可为空，若填则须符合邮箱格式 |
| amount | 记账记录 | 必填，数值大于 0，最多两位小数 |
| type | 记账记录 | 必填，只能是 income 或 expense |
| category | 记账/预算 | 必填，须在预设分类列表中 |
| recordDate | 记账记录 | 必填，格式 yyyy-MM-dd，不能是未来日期 |
| month | 预算管理 | 必填，格式 YYYY-MM |
| remark | 记账记录 | 可选，最大长度 255 个字符 |
