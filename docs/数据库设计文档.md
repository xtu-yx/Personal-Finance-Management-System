# 数据库设计文档

## 数据库概览
- 数据库名：`personal_finance`
- 字符集：utf8mb4
- 共 4 张表：user、record、budget、category

## 1. 用户表 (user)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键，自增 |
| username | VARCHAR(50) | 用户名，唯一 |
| password | VARCHAR(255) | BCrypt 加密密码 |
| email | VARCHAR(100) | 邮箱，可选 |
| create_time | DATETIME | 注册时间 |
| update_time | DATETIME | 更新时间 |

## 2. 收支记录表 (record)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键，自增 |
| user_id | BIGINT | 外键，关联 user.id |
| type | VARCHAR(10) | 类型：income（收入）/ expense（支出）|
| amount | DECIMAL(10,2) | 金额 |
| category | VARCHAR(50) | 分类 |
| remark | VARCHAR(255) | 备注，可选 |
| record_date | DATE | 记账日期 |
| create_time | DATETIME | 创建时间 |

索引：user_id、record_date、联合索引 (user_id, record_date)

## 3. 预算表 (budget)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键，自增 |
| user_id | BIGINT | 外键，关联 user.id |
| category | VARCHAR(50) | 预算分类 |
| amount | DECIMAL(10,2) | 预算金额 |
| month | VARCHAR(7) | 预算月份，格式 YYYY-MM |
| create_time | DATETIME | 创建时间 |
| update_time | DATETIME | 更新时间 |

索引：联合索引 (user_id, month)

## 4. 分类表 (category)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INT | 主键，自增 |
| name | VARCHAR(50) | 分类名称 |
| type | VARCHAR(10) | 所属类型：income / expense |

初始化数据包含 14 条常见分类。

## 表关系说明
- user 1 — N record（一个用户有多条记录）
- user 1 — N budget（一个用户可设置多个预算）
- category 仅作为字典表，不与 record 强制外键关联
